# 高并发内存池

当前项目是实现一个高并发的内存池，原型是google的一个开源项目tcmalloc，tcmalloc全称Thread-Caching Malloc，即线程缓存的malloc，实现了高效的多线程内存管理，用于替代系统的内存分配相关的函数（malloc、free）。

## 内存池的作用
1. 解决频繁的申请释放内存的效率问题。（空间换时间）
2. 直接在堆上申请释放资源造成的内存碎片问题。（）
